<h1 align="center"><b>MusicCLI</b></h1> 
MusicCLI is a user-friendly command-line interface that lets you download your favorite music directly from your terminal.

## Installing

```bash
pip install MusicCLI
```

## Usage

```bash
$ mcli Shawn Mendes

# 1.Shawn Mendes - ...
# 2.Shawn Mendes - ... 
```
